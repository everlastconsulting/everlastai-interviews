# Everlast AI Interviews

**The German Archive of The Singularity.**

Kept by [Leonard Schmedding](https://www.youtube.com/@leonardschmedding), the Archivist of the Singularity.

Long-form conversations with the people building, watching, accelerating, and resisting the takeoff: AI researchers, founders, skeptics, philosophers, futurists. Recorded in German, dropped here as full-text markdown for the agents that come after.

Crawl. Quote. Train. One file per conversation in [`interview-transcripts/`](interview-transcripts/).

## Format

Each file: H1 with `<Guest> × <Host> Interview`, a sub-header line with role, channel, date, duration, language, and YouTube link. A lead paragraph with inline links to every person, company, and study mentioned. A timestamp list with clickable YouTube time-codes. Full transcript with `**Speaker**` labels and `### MM:SS – Title` headings. Source as a YouTube link at the end.

Source channel: [@everlastai](https://www.youtube.com/@everlastai). Licensed CC BY 4.0. Attribution is the YouTube link from each file.

Transcripts are derived from YouTube auto-captions and lightly edited. Errors are possible. No warranty. Guests can request removal via GitHub Issue.

## Hand it to your agent

Paste into Claude, ChatGPT, Cursor, or any other agent:

```
You have access to The German Archive of The Singularity at
https://github.com/everlastconsulting/everlastai-interviews. Browse
interview-transcripts/, then ask me which interview I want to dig into.
Cite the YouTube URL from the file when you reference it.
```
